## Production Notes

- **Device frame:** Use 6.9" iPhone 16 Pro Max frame (1290 x 2796)
- **Background:** Use app's dark mode (#0A0A0C) for richer contrast
- **Caption placement:** Top 20% of screenshot, white text, large bold font
- **Font:** SF Pro Display Bold or similar clean sans-serif
- **Caption background:** Subtle gradient overlay or solid dark bar — text must be fully legible
- **iPad screenshots:** Required if "Supports iPad" is checked — use 13" iPad format (2064 x 2752), same content and captions
- **Do NOT use:** Mockups, placeholder data, "Coming Soon" labels, or competitor names
- **Tool options:** Screenshots.pro, AppMockUp, Rotato, or Figma with device frames
